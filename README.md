# 🏀 NBA Basketball Simulator

A realistic NBA basketball simulation game built in Python. Pick your favorite NBA teams, simulate full games, manage rosters, and experience realistic basketball with player ratings, coaching styles, and advanced game logic.

---

## Features

### 🏀 Gameplay

* Full NBA game simulation
* All 30 NBA teams
* Realistic possessions
* Four quarters plus overtime
* Live play-by-play commentary
* Team-colored text output

### 👥 Rosters

* Updated NBA rosters
* Real projected starting lineups
* Individual player ratings
* Offensive and defensive attributes
* Shooting ratings
* Playmaking ratings
* Rebounding ratings
* Injury risk system

### 📊 Statistics

* Team box scores
* Player statistics
* Quarter-by-quarter scoring
* Final game summary
* Team ratings

### 🎮 Team Selection

* Choose from every NBA team
* Color-coded team names
* Easy menu system

### 🧠 Basketball Logic

* Team ratings influence games
* Offensive and defensive matchups
* Coaching styles
* Random events for realistic outcomes
* Overtime when games are tied

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/nba-simulator.git
```

Go into the project folder:

```bash
cd nba-simulator
```

Run the game:

```bash
python main.py
```
